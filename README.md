🧠 Online Quiz Portal

A modern web-based Online Quiz Portal built using React.js that allows users to attempt quizzes across multiple subjects with a timer, automatic scoring, and interactive UI.

📌 Project Overview

The Online Quiz Portal is designed to help students practice and test their knowledge in various domains like programming, databases, and aptitude. The system presents randomized questions, tracks time, and displays results instantly.
